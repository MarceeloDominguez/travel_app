## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/travel_app.git
```

Go to the project directory

```bash
  cd travel_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
